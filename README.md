# 🚀 CareerOS AI

CareerOS AI is an AI-powered career growth assistant built with React that helps students and job seekers track goals, analyze resumes, match jobs, and receive personalized career guidance.

## 🌐 Live Demo

https://careeros-ai.netlify.app/

## 📂 GitHub Repository

https://github.com/debugwithyash/careeros-ai

---

## ✨ Features

### 🎯 Career Goal Tracker

* Save career goals
* Track career direction
* Persistent storage using LocalStorage

### 🧠 Skills Memory

* Store and manage skills
* Skills used across modules

### 📄 Resume Analyzer

* Upload resume
* Extract skills
* Generate resume score

### 💼 Job Matcher

* Paste any job description
* Calculate skill match percentage
* Display missing skills
* Recommend learning path

### 🤖 AI Career Advisor

* AI-powered roadmap generation
* Internship recommendations
* Project suggestions
* Personalized career guidance

### 🐙 GitHub Profile Integration

* Fetch GitHub profile data
* Display repositories and profile information

### 📈 Career Readiness Dashboard

* Dynamic career score
* Progress visualization
* Goal tracking metrics

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* Vite
* Tailwind CSS
* OpenRouter API
* LocalStorage

---

## 🚀 Installation

```bash
git clone https://github.com/debugwithyash/careeros-ai.git

cd careeros-ai/client

npm install

npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
VITE_OPENROUTER_API_KEY=your_api_key_here
```
